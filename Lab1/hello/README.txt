HELLO
=====
	
ОПИСАНИЕ
	Программа предназначена для вывода приветствия на экран.

СБОРКА
	Для сборки программы необходимо ввести
	$ make
	или
	$ make all

КОД
	В директории src/ содержатся файлы с исходным кодом программы.

ЗАДАНИЯ
	В директории doc/ содержатся задания для студентов (файл
	tasks.txt), а также краткое руководство по созданию и
	использованию Makefile-ов и утилиты GNU make (mini_make.txt).
